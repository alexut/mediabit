Icon Usage Guide

.icon-set: Width 4rem, Stroke-width 0
.icon-small: Width 2rem
.icon-large: Width 8rem
.icon-primary: Stroke and fill color primary
.icon-animate: Animation on hover, darker primary color
.icon-secondary: Stroke and fill color secondary
.icon-info: Stroke and fill color info
.icon-light: Stroke and fill color light
.icon-warning: Stroke and fill color warning
